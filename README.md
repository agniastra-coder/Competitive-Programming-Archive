# Competitive Programming Archive

A curated archive of optimized C solutions for abstract algorithmic challenges and competitive programming logic puzzles. 

This repository highlights a focus on edge-case handling, mathematical proportionality, and grid-based logic evaluation.

### 🧠 Logic & Algorithms Covered
* **Coordinate Parity & XOR Logic:** Evaluating chess-grid coordinates to determine color states and XOR outcomes.
* **Proportionality Validation:** Cross-multiplication logic for verifying the equivalence of multiple ratios.
* **1D Range Inclusion:** Algorithms for segment boundary checking and bounding-box validation.

### 🛠️ Tech Stack
* Language: C
* Focus: Algorithmic Problem Solving, Logic Optimization
